FAQ

C'est quoi Dimtitou ?
Dimtitou est un forum centré sur l'anonymat, visant a fournir des cercles de discussions a toutes personnes cherchant un moyen de s'exprimer librement sans laisser de trace.
Comment puis-je créer un compte ?
Afin de garantir l'anonymat a nos utilisateurs, il n'est pas possible de créer de compte sur dimtitou.
Que signifie le logo du site ?
Le logo du site dimtitou est une spirale(en réference au systeme d'exploitation Debian Linux)noir avec des points blanc, le noir et le blanc etant les couleurs préféré de l'administrateur.

dimtitou étant un site sous license GPL, le code source de ce dernier est entièrement accessible via ce lien.
